# 🏦 Loan Risk Analysis

Predicts applicant risk level (low / medium / high) from financial
and demographic features using machine learning.

## 📂 Notebooks
| # | Notebook | Description |
|---|----------|-------------|
| 1 | `01_data_collection.ipynb` | Load, inspect & enrich dataset |
| 2 | `02_eda_visualization.ipynb` | EDA, charts & correlation analysis |
| 3 | `03_ml_model.ipynb` | Feature engineering, model training & evaluation |

## 🗃️ Dataset
`loan_risk_data.csv` — 2,000 applicants, 31 features.

## 🚀 Quick Start
```bash
pip install -r requirements.txt
jupyter notebook
```
Run the notebooks in order: 01 → 02 → 03.
